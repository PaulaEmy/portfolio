Site de avaliação de projetos para Feira Técnica

Projeto feito para meu TCC do Colégio UNIVAP Centro.
Tem como objetivo a criação de um site com a intenção de facilitar a organização dos projetos e permitir que professores e visitantes possam avaliar digitalmente os trabalhos da feira técnica.

Participei do desenvolvimento do design do site e de seus requisitos funcionais e não funcionais.

Desafios: Criação de crachá e gráfico no site.
Solução: Utilizei chart.js para criação dos gráficos e criei funções para pegar as informações para o crachá.

Tecnologias usadas:

  • HTML

Primeira API para a FATEC.
